## Getting Started

You can either click [`Use this template`](https://github.com/agustinusnathaniel/rsbuild-react-base/generate) button on this repository and clone the repo or use npx degit like so:

```bash
npx degit agustinusnathaniel/rsbuild-react-base <app_name>
```

Then, run the development server:

```bash
pnpm start
```
